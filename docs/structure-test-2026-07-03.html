Honest limits of the test

  • Cohort shapes differ: v3 sessions were hub/supervisor-shaped; controls carried heavy delegated-worker traffic. Only 2 Claude v2 closures were cleanly segmentable; Codex counts bundle resume with closure.
  • ttrpg's "v2" closures were mostly performed by v3 hub sessions and touched only 3 of 6 lanes (~9 calls) — lane discipline slips even when v2 is actively maintained, which itself supports the v3 thesis.
  • The v2 quiz baseline was measured at its freshly-consolidated best (commit 5f81c40); its steady state carried 40 overlap warnings, so the comparison flatters v2.
  • Quiz Q5 pins to the current latest release (v0.0.18 at rerun vs v0.0.14 at baseline) — same slot in the pre-registered template.
